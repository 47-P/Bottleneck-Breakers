Project Overview

Our project tackles the time-consuming challenge of summarizing large collections of academic papers. By leveraging Large Language Models (LLMs), we've developed a sophisticated two-stage process that first generates individual summaries for each paper, then aggregates them into a cohesive, comprehensive document.

🔍 Intelligent Parsing
Advanced text extraction with support for multiple formats including PDF, DOCX, HTML, and more with OCR capabilities
⚡ Parallel Processing
Multi-threaded chunk processing with linear scaling up to 4 workers for optimal performance
🛡️ Secure & Private
Offline mode available with no data transmission, plus PII sanitization for online processing
🎯 Flexible Output
Configurable summary lengths (short, medium, detailed) with multiple export formats
🖥️ Multi-Interface
GUI, CLI, and web application interfaces for different user preferences and workflows
🔧 Cross-Platform
Windows, macOS, and Linux support with Docker containerization for easy deployment
